AM6 Runtime Keystore¶

Introduction¶

This chapter provides information on the size of the Runtime Keystore as it is configured for AM6 SoC.

Size of Runtime Keystore¶

Runtime keystore size is confirugred during System Firmware build and is defined for this device to be the following:

#define CONFIG_SECURITY_KEYSTORE_SIZE (0x2800U)
