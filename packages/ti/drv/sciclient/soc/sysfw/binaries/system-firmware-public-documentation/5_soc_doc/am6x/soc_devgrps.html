AM6 Device Group descriptions¶

Introduction¶

This chapter provides information of specific device groups(DEVGRP) that are permitted in the AM6 SoC. These device groups are more specific grouping of devices and resources used optionally to have staged init sequence control of the SoC.

Enumeration of Device group IDs¶

SoC Devgrp ID Generic Devgrp ID Flag Value Description
MCU_WAKEUP DEVGRP_00 0x1U << 0U Set of MCU domain peripherals to initialize at boot and use during runtime. Utilize this device group of MCU/Wakeup domain peripherals to achieve a faster boot sequence
MAIN DEVGRP_01 0x1U << 1U Set of Main domain peripherals to initialize at boot and use during runtime. Utilize this device group of non-critical peripherals for advanced use cases.
