=============================
AM6 Secure Proxy Descriptions
=============================

.. _soc_doc_am6_public_sproxy_desc_intro:

Introduction
============

This chapter provides information of Secure Proxies and communication paths
that are permitted in the am6 SoC. These host IDs represent processing entities
(or PEs) which is mandatory identification of a Host in a processor. See
PE/Host documentation for further information
