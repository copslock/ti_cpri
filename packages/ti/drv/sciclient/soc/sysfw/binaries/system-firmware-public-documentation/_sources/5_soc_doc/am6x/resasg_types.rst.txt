=============================================================
AM6 Board Configuration Resource Assignment Type Descriptions
=============================================================

Introduction
============

This chapter provides information of Board Configuration resource assignment
type IDs that are permitted in the AM6 SoC. These resource type IDs represent
AM6 resources that can have ranges assigned to SoC processing entities (or PEs).
