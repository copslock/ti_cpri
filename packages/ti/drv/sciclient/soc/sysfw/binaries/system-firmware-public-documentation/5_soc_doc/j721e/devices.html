J721E Devices Descriptions¶

Introduction¶

This chapter provides information on Device IDs that are permitted in the j721e SoC. The device IDs represent SoC subsystems that can be modified via DMSC TISCI message APIs. Some Secure, Power, and Resource Management DMSC subsystem TISCI message APIs define a device ID as a parameter allowing a user to specify management of a particular SoC subsystem.
