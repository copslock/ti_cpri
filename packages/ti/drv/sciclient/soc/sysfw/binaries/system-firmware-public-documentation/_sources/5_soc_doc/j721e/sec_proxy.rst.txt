===============================
J721E Secure Proxy Descriptions
===============================

.. _soc_doc_j721e_public_sproxy_desc_intro:

Introduction
============

This chapter provides information of Secure Proxies and communication paths
that are permitted in the J721E SoC. These host IDs represent processing entities
(or PEs) which is mandatory identification of a Host in a processor. See
PE/Host documentation for further information

.. _soc_doc_j721e_public_sproxy_desc_sproxy_list:
