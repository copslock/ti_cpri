The example demonstrates CPPI API usage, channel and flow resource allocation on multiple cores. 
The example MUST be loaded on all 4 cores. To change the number of cores this example is run on,
modify the core configuration in the sample.cfg file. The example is provided for both Big and Little Endian. 

Check the release notes for prerequisites, version information and steps on how to run examples
